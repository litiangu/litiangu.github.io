发表于 2019-07-29 分类于 Ubuntu ， Ubuntu指令

ubuntu经常使用到的命令

  1. ps aux **—列出系统上运行的所有进程的详细信息。 这包括用户、进程IDPID**和过程的名称。 使用此可查看进程列表中并在必要时您可以终止不必要的或已停止进程。
  2. kill pid-PID是数字的引用有问题的进程。 您应获取从命令(如ps auxPID。 如果拒绝中止的进程,您也可以指定kill-9PID应终止进程以任何方式。
  3. 丢失自己的目录中? 键入pwd打印工作目录。

要计算您的磁盘空间吗? df-h可向您提供快速的图。
要计算某个文件夹或文件的大小? du-cksh target_name将执行完全一致的。 要计算当前文件夹的大小? du-cksh。
想要将文件标记为可执行文件? chmod+x filename将执行该操作。

  • 删除桌面上的Amazon链接

    sudo apt-get remove unity-webapps-common
    sudo apt remove unity-webapps*
    sudo apt remove purge unity

  • 关闭系统更新，修改配置文件,所有值改为“0”

    sudo gedit /etc/apt/apt.conf.d/10periodic

  • 安装.deb

    安装命令
    sudo dpkg -i <安装包名称>
    如果有依赖故障，用以下命令修复故障
    sudo apt-get install -f

  • 卸载自动的libreoffice

    sudo apt-get remove –purge libreoffice*

    apt-get remove 软件包名称：

删除已安装的软件包（保留配置文件）。

apt-get –purge remove 软件包名称：

删除已安装包（不保留配置文件)

apt-get autoremove:

删除为了满足其他软件包的依赖而安装的，但现在不再需要的软件包。


  • > sudo apt-get clean # 删除安装了的软件包的安装包（如果你确定这些安装包不会再用）。 删除包缓存中的所有包。

sudo apt-get autoremove # 删除为了满足其他软件包的依赖而安装的，系统会自动卸载这些不再需要的软件包
